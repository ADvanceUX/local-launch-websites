# Local Launch Websites

Modern React and Vite website for an Ireland-based website creation business helping small and local businesses get online.

## Run Locally

```powershell
npm.cmd install
npm.cmd run dev
```

## Build

```powershell
npm.cmd run build
```

## Lead Form

The contact form submits enquiries through Formspree. The endpoint is configured in `src/App.jsx`.
